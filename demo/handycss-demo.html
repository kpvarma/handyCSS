Handy CSS - Documentation

Overview

Handy CSS is a collection of classes which are quite handy for bootstrapping a project. It has 3 components.

  1. Basic layouting classes
  2. Boxes for displaying feeds, articles and blog posts.
  3. A collection of 20+ Backdrops
